    📖 How to export bookmarks from your browser:

    1. Chrome / Edge: Bookmarks menu → Bookmark manager → ⋮ → Export bookmarks
    2. Firefox: Bookmarks → Manage Bookmarks → Import & Backup → Export Bookmarks to HTML
    3. Safari: File menu → Export Bookmarks…
    4. Firefox (JSON): Manage Bookmarks → Import & Backup → Backup… (saves as .json)

    🔧 How to show your bookmark bar:

    1. Chrome / Edge: Ctrl+Shift+B (Win) or ⌘+Shift+B (Mac)
    2. Firefox: View → Toolbars → Bookmarks Toolbar
    3. Safari: View → Show Favorites Bar
